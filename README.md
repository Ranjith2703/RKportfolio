# 💼 Ranjith Kumar A - Personal Portfolio

A modern and interactive personal portfolio website showcasing my projects, technical skills, certifications, achievements, coding profiles, and professional journey as a Software Developer.

🔗 **Live Demo:** https://ranjithak.vercel.app/

---

## ✨ Features

- 🚀 Animated preloader
- 🎨 Modern dark-themed UI
- 🌌 Interactive animated background
- 📱 Fully responsive design
- 📄 Resume download
- 💼 Professional experience section
- 🎓 Education timeline
- 🛠️ Technical skills showcase
- 💻 Live LeetCode profile integration
- 📂 Project portfolio with live demos
- 🏆 Certifications section
- 📞 Contact information
- 🔗 Social media integration
- 👀 Visitor counter
- ⚡ Smooth scrolling navigation
- 📱 Mobile-friendly navigation
- ✨ Modern animations and transitions
